# Sonic™ / Project Apollo

Sonic is the first production department of Mobius OS and the reference implementation for Operation Atlas.

## Mission

Prove Mobius OS Alpha through real audio-production work.

## Platform Role

Sonic is not a standalone assistant. Sonic is a department that plugs into:

- Atlas Core
- Organization Brain
- Mobius OS Core
- Event Bus
- Permission Engine
- Automation Engine

## Alpha Capabilities

Sonic / Project Apollo must prove:

- Plugin discovery
- Plugin inventory search
- Artist Brain profiles
- Session Brain profiles
- Reference analysis workflow
- Pro Tools workflow guidance
- SoundFlow automation planning and execution where connected
- Session memory write-back into the Organization Brain

## Operating Rule

Sonic does not own organizational knowledge. It reads from and contributes to the Organization Brain through Atlas Core.

## Demo Standard

The Alpha workflow should allow Michael to launch Mobius, route to Sonic, load an artist profile, analyze or describe a reference, recommend a chain using the real plugin inventory, prepare approved automation, and save the session outcome.
